# Musical Inventory

Express + EJS + PostgreSQL inventory app.

## Requirements
- Node.js 18+
- PostgreSQL

## Setup
1. Install dependencies:
   npm install

2. Create database (example):
   createdb inventory_db

3. Configure .env with DATABASE_URL.

4. Initialize schema:
   node db/initDB.js

5. Start the app:
   npm run dev
   or
   npm start

## Folders
- controllers
- routes
- views
- db
- public
